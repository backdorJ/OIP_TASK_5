## Векторный поиск по TF‑IDF

Реализован в скрипте `vector_search.py`. Векторный поиск работает по TF‑IDF в пространстве лемм:

- для каждого документа заранее посчитан вектор `doc_vectors[doc_id] = {лемма: tfidf}`;
- запрос пользователя также превращается в вектор TF‑IDF по леммам;
- релевантность документа запросу оценивается по **косинусному сходству**:
  \[
  \text{sim}(\vec{q}, \vec{d}) = \frac{\vec{q} \cdot \vec{d}}{\|\vec{q}\| \cdot \|\vec{d}\|}
  \]
- документы сортируются по убыванию сходства, выводится топ‑N.

### Запуск векторного поиска

Сначала посчитать TF‑IDF (если ещё не посчитан):

```bash
python3 build_tfidf.py
```

Затем запустить векторный поиск:

```bash
python3 vector_search.py "Клеопатра Цезарь Антоний"
```

Или без аргумента — запрос будет предложено ввести с клавиатуры:

```bash
python3 vector_search.py
```

Программа выведет топ‑10 документов с наибольшим косинусным сходством с запросом (doc_id и URL из `index.txt`).
